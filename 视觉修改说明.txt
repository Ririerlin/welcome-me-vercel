视觉修改说明

这版已经完成以下改动：
1. 首屏加入正式活动主视觉图，减少空白和 Demo 感。
2. 登录页加入活动海报图，进入前就能看到活动场景。
3. 首页顶部加入活动海报卡，强化“正式小程序首页”的第一眼印象。
4. 新增 A1 馆导览图，让现场服务更真实。
5. 展品图从外链换成本地 SVG 图片，部署更稳定。
6. 参会者头像增加本地统一风格头像，减少 emoji 半成品感。
7. 精简首屏文案，让说明更像正式活动产品。
8. 修正部分不标准的 Tailwind 色值类名，降低部署样式异常风险。

使用方法：
1. 解压压缩包。
2. 把里面的项目文件上传或覆盖到 GitHub 仓库。
3. 提交后等待 Vercel 自动部署。

我已经检查过：npm run build 和 npm run lint 均通过。
