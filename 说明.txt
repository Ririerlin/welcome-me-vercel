欢迎ME 服务端卡片布局与夜间模式修复补丁

修复内容：
1. 给后台增加 admin-console 样式作用域，避免前台小程序全局 CSS 污染后台。
2. 给用户端增加 mini-app-theme 标记，方便前后台样式隔离。
3. 修复左侧菜单卡片结构：字号、图标、间距恢复到更标准的后台导航。
4. 修复“参会用户资料”卡片：不再像大胶囊，标签与签到状态不再挤压。
5. 修复搜索框交互格式。
6. 增强后台夜间模式可读性，提亮灰字、修正深色卡片和输入框。

安装方式：
unzip -o ~/Downloads/welcome-me-admin-card-dark-fix.zip
npm run build
git add .
git commit -m "fix admin card layout and dark mode readability"
git push

如果想回滚：
git revert HEAD --no-edit
git push
