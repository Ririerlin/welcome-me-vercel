欢迎ME 服务端系统设置真实网站化补丁

本补丁基于上一版 admin-card-dark-fix 做增量优化：
1. 将“系统设置”从简单说明卡片改成真实 SaaS 后台设置页
2. 新增活动基础配置、品牌视觉系统、报名与用户资料、权限角色、通知自动化、数据集成、报告导出配置、高级设置与安全区
3. 继续保留后台样式隔离与夜间模式可读性修复
4. 保留 /admin 后台入口，不影响用户端小程序

安装方式：
unzip -o ~/Downloads/welcome-me-admin-real-settings-upgrade.zip
npm run build
git add .
git commit -m "upgrade admin settings page"
git push

部署后打开：
https://welcome-me-vercel.vercel.app/admin?v=27
