本补丁修复：
1. 在“我的”页面增加：更换头像、切换账号、退出登录。
2. 注册页重新加入“上传头像”功能，上传后会作为个人头像显示。
3. 个人资料页头像支持上传图片，并保存到浏览器本地 localStorage。
4. 退出登录后不会继续自动进入账号；切换账号会返回登录页。

使用方式：
在项目根目录执行：
unzip -o ~/Downloads/welcome-me-account-avatar-bugfix.zip
npm run build
git add .
git commit -m "fix account switching and avatar upload"
git push

如果 GitHub 网络报错，等网络恢复后再次执行：
git push
