欢迎ME 服务端真实交互补丁

本补丁只覆盖：
- src/components/AdminDashboard.tsx

修复/新增：
1. 系统设置页的输入框、开关、上传、角色人数、报告模块、导出格式都可以真实交互
2. 设置内容会保存到浏览器 localStorage
3. 活动基础信息可编辑
4. 品牌 Logo 文案、主题色、封面图、报告背景可修改
5. 报名资料开关、通知开关、集成状态开关可切换
6. 权限角色人数可加减
7. 报告模块可勾选/取消，导出格式可切换
8. 高级设置里的导出快照、重置本机演示数据、管理员访问控制都能点击生效
9. 其他后台模块也保留可交互：签到、关系确认、提问审核、议程直播/推荐切换、导出报告数据等

说明：
这是前端本地可交互版本，数据保存在当前浏览器。真正多设备同步需要后续接 Supabase / Firebase / 数据库。

安装：
unzip -o ~/Downloads/welcome-me-admin-real-interactions.zip
npm run build
git add .
git commit -m "make admin dashboard interactive"
git push
